# llm-hiring-pipeline
# 🚀 GenAI Hiring Workflow

An AI-powered hiring automation system that streamlines resume screening using Large Language Models (LLMs).

---

## 📌 Overview

This project automates the initial candidate evaluation process by extracting resume data, analyzing it using AI, and generating structured insights such as scores, decisions, and communication drafts.

---

## ⚙️ Features

- 📄 Resume Parsing (PDF + OCR support)
- 🤖 AI-based Candidate Evaluation (LLM)
- 📊 Scoring & Shortlisting System
- 📧 Email & WhatsApp Communication Drafts
- 🗂 Data Storage (CSV-based)

---

## 🧠 Tech Stack

- Python
- Streamlit
- OpenRouter API (LLM)
- pdfplumber
- Tesseract OCR
